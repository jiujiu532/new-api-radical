# 🎉 v1.2.0 - 模型重定向模板管理

## ✨ 新增功能

### 🔄 模型重定向模板管理

渠道编辑页面全新升级，大幅提升模型重定向配置效率！

| 功能 | 说明 |
|------|------|
| **批量前缀** | 一键为所有映射添加/删除前缀（如 `openai/`） |
| **批量后缀** | 一键为所有映射添加/删除后缀（如 `-latest`） |
| **一键重定向** | 根据模型列表自动生成重定向映射 |
| **模板导入/导出** | 支持 JSON 文件导入导出，方便跨实例迁移 |
| **恢复默认** | 一键恢复内置默认模板 |

### 📦 内置模型重定向模板

预置常用模型提供商的重定向规则：

- **Claude** → OpenAI 格式
- **Gemini** → OpenAI 格式  
- **OpenAI o 系列** → 标准格式
- **DeepSeek** → OpenAI 格式
- **Grok** → OpenAI 格式
- **Llama** → OpenAI 格式
- 更多...
